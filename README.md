🍽️Flavourscapes Restaurant Website 

📌 Project Overview  
This project is a responsive Restaurant Website developed using HTML5, CSS3, and Bootstrap 5.  
It showcases a variety of food categories such as starters veg and non-veg, Veg and Non-Veg currys, Biryanis, Sweets,frys and Drinks using an attractive card-based layout.  
The website is designed to provide a clean, user-friendly, and visually appealing interface for customers to explore different menu items.

🚀 Features  
- Responsive multi-section restaurant website  
- Attractive food menu layout using Bootstrap cards  
- Separate categories for:
  - Veg
  - Non-Veg
  - Biryanis
  - Curries
  - Sweets
  - Drinks etc..
- Smooth hover effects on menu cards  
- Clean and modern UI design  
- Mobile-friendly responsive layout  
- Easy navigation with navbar sections  

🛠️ Technologies Used  
- HTML5  
- CSS3  
- Bootstrap 5  
- Google Fonts  

📂 Project Structure  
- index.html – Main homepage  
- style.css – Styling file  
- images/ – Food item images  

✨ UI Highlights  
- Beautiful background design  
- Responsive Bootstrap grid system  
- Card hover animations  
- Category-wise food display  
- Modern restaurant theme layout  

🎯 Purpose of the Project  
The main goal of this project is to practice responsive web design and create a visually appealing restaurant menu website using HTML, CSS, and Bootstrap.
